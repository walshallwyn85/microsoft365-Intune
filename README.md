# Microsoft Entra ID + Intune Home Lab

## Overview
This home lab was built to practice real-world IT support and endpoint administration tasks using Microsoft Entra ID and Microsoft Intune.

## Objectives
- Learn Microsoft Entra ID user and device management
- Practice Intune enrollment for Windows and iOS devices
- Configure compliance policies and configuration profiles
- Simulate common IT support issues related to enrollment and access
- Document troubleshooting steps and resolutions

## Technologies Used
- Microsoft Entra ID
- Microsoft Intune
- Windows 11
- iPhone / iPad
- Microsoft Authenticator
- Company Portal
- Virtual machines
- Windows Server (if used in your lab)

## Lab Tasks Completed
- Created test users in Microsoft Entra ID
- Assigned licenses
- Tested device registration and enrollment
- Configured compliance policies
- Created configuration profiles
- Practiced troubleshooting enrollment issues
- Tested access through Company Portal and Microsoft Authenticator

## Troubleshooting Scenarios
- License assignment cannot be done for user with invalid usage location
- MDM authority not configured / unknown
- Company Portal enrollment issues
- Microsoft Authenticator redirect during login
- Device not appearing in Intune
- iOS enrollment errors

## Key Skills Demonstrated
- Identity and access management
- Endpoint management
- Mobile device enrollment
- Troubleshooting methodology
- Documentation
- IT support lab simulation

This is a self-built home lab created for learning and hands-on practice in enterprise support technologies.
